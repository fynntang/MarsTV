# Web App (Next.js 15)

Next.js 15 + App Router + React 19 + TypeScript + Tailwind CSS 4.

详见根 [CLAUDE.md](../../CLAUDE.md)。

## M1 脚手架待补

此目录当前只是占位,M1 阶段需执行:

```bash
# 在根目录运行
pnpm dlx create-next-app@latest apps/web \
  --ts --app --tailwind --src-dir --import-alias "@/*" --use-pnpm
```

然后按计划接入 `packages/core`、`packages/ui-web`、`packages/config`。
